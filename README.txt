qwertonic-py
a prototype implementation of the qwertonic keyboard
(http://www.qwertonic.com/)
version 0.2

requires pyo
https://code.google.com/p/pyo/

files:
    qwertonic.py
    music.py
    resources/qwertonic-illustration.gif

operation:
    > py qwertonic.py
    then type to play notes
    close window when finished

flags:
    > py qwertonic.py --help
    then read what follows
